# CodeAlpha_PhishingAwarenessTraining

> **CodeAlpha Cybersecurity Internship — Task 2**
> An interactive phishing awareness training module built with HTML, CSS, and JavaScript.

---

## 📌 About the Project

This is an interactive online training module that educates users about phishing attacks — how they work, how to recognize them, and how to stay safe. It covers all major types of phishing with real-world examples and an interactive quiz.

---

## ✅ Features

- **6 Types of Phishing** — each with a description and real-world example
- **Clickable Buttons** — click any type to learn about it and see its example
- **Real Examples** — fake emails, SMS, phone calls, and websites shown visually
- **Red Flags Highlighted** — shows exactly what to look for in each attack
- **Best Practices** — 7 tips to stay safe
- **Interactive Quiz** — 3 questions with instant feedback and final score

---

## 🦠 Phishing Types Covered

| Type | Description |
|---|---|
| 📧 Email Phishing | Fake emails pretending to be from banks or companies |
| 📱 Smishing | Phishing via SMS text messages |
| 📞 Vishing | Phishing via phone calls (voice) |
| 🎯 Spear Phishing | Targeted attacks using your personal information |
| 🐋 Whaling | Targeting CEOs and executives for large fraud |
| 🌐 Fake Website | Fake login pages that steal your credentials |
